# Installation

Spectrl is installed via npm or used via `npx`.

## npx (recommended)

No install required - just run:

```bash
npx spectrl
```

## Global install

If you prefer a persistent install:

```bash
npm install -g spectrl
```

## Verify installation

```bash
npx spectrl --version
```

## Updating

If installed globally:

```bash
npm update -g spectrl
```
